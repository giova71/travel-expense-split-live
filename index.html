Spese vacanza live

Una mini web app condivisa, pensata per funzionare da link unico su due telefoni.

Versione condivisa

Questa versione usa Firebase Firestore come backend realtime. Tu e la tua compagna aprite lo stesso link, impostate lo stesso ID viaggio e vedete le spese aggiornarsi quasi in tempo reale.

SyncRealtime
AccessoLink unico
Split50 / 50

Setup rapido

1. Crea un progetto Firebase e abilita Firestore.
2. Inserisci la config nel blocco firebaseConfig.
3. Pubblica il file su GitHub Pages, Netlify o Azure Static Web Apps.
4. Condividi il link e usate lo stesso ID viaggio.

Connessione

Inserisci la tua config Firebase prima di pubblicare l'app.
